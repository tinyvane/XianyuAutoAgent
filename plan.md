# XianyuAutoAgent - 闲鱼 AI 自动客服 - 项目规划

## 项目概述
Python 3.8+ + Node.js 18+ + LLM（GPT/Claude等）构建的闲鱼平台7×24小时自动值守 AI 解决方案，支持智能议价、上下文感知对话、多专家协同决策，包含网络搜索集成和完整日志监控。

---

## 2026-03-03 Claude 建议

### 与相关项目关系
与 `20251017-xianyuAutoTakeOver` 和 `XianYuApis` 高度相关：
- **此项目（XianyuAutoAgent）**：AI 对话引擎，多专家协同
- **xianyuAutoTakeOver**：更注重自动发货等自动化操作
- **XianYuApis**：底层 API 接口封装

建议三者合并或明确分工（API层→业务层→AI层）。

### 功能建议

#### 短期（智能化提升）
1. **多专家协同优化**
   - 专家路由更准确（意图分类模型）
   - 专家间知识共享（避免重复问题）
   - 专家置信度（不确定时转人工）
   - 专家性能监控（哪个专家处理效果最好）

2. **议价策略**
   - 商品底价自动从描述/成本推算
   - 让步幅度基于商品滞销时间动态调整
   - 批量购买自动优惠
   - 快速成交优惠（限时特价）

3. **上下文管理**
   - 长期记忆（该买家以前问过什么）
   - 商品相关知识库（常见问题自动匹配）
   - 多轮对话连贯性优化

4. **人工干预**
   - 指定关键词触发人工接管
   - 实时对话监控界面
   - 一键接管/还给机器人

#### 中期
1. **商品智能运营**
   - 商品描述 AI 优化建议
   - 定价策略推荐（参考同类商品）
   - 商品标签自动生成
   - 上架最佳时间建议

2. **数据分析**
   - 对话成功率漏斗
   - 议价分析（买家出价分布）
   - 最常见问题统计
   - 收入数据报告

3. **多平台扩展**
   - 转转平台支持
   - 得物平台支持
   - 拼多多二手频道

#### 长期
1. **AI Agent 进化**
   - 主动出击（自动联系潜在买家）
   - 定价动态调整（无人问津时自动降价）
   - 竞品监控（类似商品价格追踪）

2. **SaaS 模式**
   - 多卖家账号托管
   - 按成交量收费

3. **整合方向**
   - 与 `XianYuApis` 合并为底层
   - 与 `xianyuAutoTakeOver` 合并为完整方案

### 技术建议
- 使用本地轻量模型（如 Qwen-7B）作为主力（减少 API 成本）
- 对话缓存（相似问题复用答案）
- 限流控制（避免账号被平台检测）
- 添加 A/B 测试（对比不同回复策略效果）
